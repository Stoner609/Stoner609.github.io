指令操作刪除遠端遠程分支
<!-- more -->

### 刪除本機端的分支 (Delete Local Branch)
```bat
git branch -d <branch_name>
git branch -D <branch_name>
```

### 刪除遠端分支
```bat
git push <remote_name> :<branch_name>
git push <remote_name> --delete <branch_name>
```
